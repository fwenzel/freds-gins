# Mr Fred's Gins

by `<mrfred@gins.pw>`
* https://gins.pw
* https://twitter.com/mrfredsgins

A modest collection of the finest thing juniper berries can become when they grow up.

To run:
* `bundle install --path vendor/bundle`
* `bundle exec jekyll serve`
